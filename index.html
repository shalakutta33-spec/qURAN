🔌 API Usage

Download File

GET /mushaf-layout/{filename}

Download a specific Mushaf layout database file.

Example Request:

curl -O https://quran-api-files.vercel.app/mushaf-layout/digital-khatt-15-lines.db

Flutter/Dart Example:

import 'package:dio/dio.dart'; final dio = Dio(); await dio.download( 'https://quran-api-files.vercel.app/mushaf-layout/digital-khatt-15-lines.db', '/path/to/save/digital-khatt-15-lines.db', onReceiveProgress: (received, total) { final progress = received / total; print('Download progress: ${(progress * 100).toStringAsFixed(0)}%'); }, );

Response Headers

Header Value Description
Content-Type application/octet-stream Binary file download
Content-Length {file_size} File size in bytes
Cache-Control public, max-age=31536000 Cache for 1 year
Access-Control-Allow-Origin * CORS enabled for all origins

🗄️ Database Schema

Mushaf Layout Tables

-- Main verses table CREATE TABLE verses ( id INTEGER PRIMARY KEY, verse_key TEXT, -- Format: "surah:ayah" (e.g., "1:1") surah INTEGER, -- Surah number (1-114) ayah INTEGER, -- Ayah number text TEXT -- Arabic text in specified script ); -- Example data: -- id: 1, verse_key: "1:1", surah: 1, ayah: 1, text: "بِسْمِ اللّٰهِ..."

Translation Tables

-- Translations table CREATE TABLE translations ( id INTEGER PRIMARY KEY, verse_key TEXT, -- Format: "surah:ayah" surah INTEGER, ayah INTEGER, text TEXT, -- Translated text language TEXT -- Language code (bn, en, etc.) );

📱 Flutter Integration

1. Add Dependencies

# pubspec.yaml dependencies: dio: ^5.4.0 sqflite: ^2.3.0 path_provider: ^2.1.2

2. Download Service

class QuranDownloadService { static const String baseUrl = 'https://quran-api-files.vercel.app'; Future downloadMushafLayout(String filename) async { final dio = Dio(); final dir = await getApplicationDocumentsDirectory(); final savePath = '${dir.path}/databases/$filename'; await dio.download( '$baseUrl/mushaf-layout/$filename', savePath, onReceiveProgress: (received, total) { final progress = (received / total * 100).toStringAsFixed(0); print('Downloading: $progress%'); }, ); } }

3. Use Downloaded Database

// Open downloaded database final db = await openDatabase(savePath); // Query verses final verses = await db.query( 'verses', where: 'surah = ?', whereArgs: [1], // Surah Al-Fatiha );

⚠️ Limits & Considerations

Vercel Free Tier Limits:

  • Bandwidth: 100 GB/month
